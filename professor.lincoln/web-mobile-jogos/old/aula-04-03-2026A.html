O Esqueleto do Mundo Digital

O HTML não é uma linguagem de programação, mas de marcação. Ele define a intenção e a hierarquia do conteúdo.

"O HTML5 funciona como a fundação de qualquer projeto web, estabelecendo a hierarquia que será interpretada pelos navegadores." — Alves (2019)

Por que Semântica?

  • Acessibilidade: Permite que leitores de tela entendam a página.
  • SEO: Motores de busca (Google) indexam melhor seu conteúdo.
  • Manutenção: Código limpo e legível para o desenvolvedor.
<!-- Estrutura Semântica -->
<header>Cabeçalho</header>
<main>Conteúdo Principal</main>
<footer>Rodapé</footer>

Glossário Técnico

<article>: Conteúdo independente (ex: post de blog).

<section>: Agrupamento temático de conteúdo.

<aside>: Conteúdo relacionado lateralmente (sidebars).

<nav>: Conjunto de links de navegação.

"A semântica do HTML é o alicerce para um design de navegação que reduza a carga cognitiva do usuário." — Kalbach (2009)

IHC e Mobile First

A Interação Humano-Computador ganha vida nos formulários. Tipos de input específicos melhoram a usabilidade no mobile:

<input type="email"> <!-- Abre teclado com @ -->
<input type="tel"> <!-- Abre teclado numérico -->
<label>Nome:</label> <!-- Aumenta área de clique -->

Segundo Mota et al. (2023), a acessibilidade tecnológica é uma obrigação no design de interfaces modernas.
