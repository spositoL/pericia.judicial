COMO USAR CÓDIGO REUTILIZÁVEL
================================================================================

Para ATIVAR CRO em qualquer página nova:

PASSO 1: Adicionar no HEAD (entre <head> e </head>)
   <link rel="stylesheet" href="cro-system.css">

PASSO 2: Adicionar antes de </body>
   <script>
      const croConfig = {
          enableProgressBar: true,
          enableFloatingBar: true,
          enableExitIntent: true,
          floatingBarDelay: 20000,
          floatingBarMessage: 'SEU TEXTO AQUI',
          floatingBarUrgency: 'SEU URGENCY AQUI'
      };
   </script>
   <script src="cro-system.js"></script>

PASSO 3: Customizar copy para página

PRONTO! CRO ativo em 3 minutos.
